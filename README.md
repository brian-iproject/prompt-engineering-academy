# Prompt Engineering Academy

Образовательный портал по промпт-инжинирингу с интерактивными упражнениями прямо в тексте статей.

## Технологии

| Технология | Почему |
|---|---|
| **Vite + Vue 3** | Composition API, реактивность без лишнего кода, быстрая сборка |
| **Vue Router** | SPA-навигация с ленивой загрузкой страниц |
| **SCSS + BEM** | Поддержка переменных и вложенности, предсказуемое именование классов |

## Компоненты упражнений

### Базовые (`src/components/quiz/`)

| Компонент | Что проверяет | Особенности |
|---|---|---|
| `TrueFalse.vue` | Оценка утверждения | Детальное объяснение в любом случае |
| `MultipleChoice.vue` | Один или несколько ответов | Поддержка частично правильных ответов, адаптивный badge |
| `FillTheBlank.vue` | Вписать нужное слово | Нормализация (регистр, ё→е), список альтернативных ответов |
| `MatchPairs.vue` | Соединить пары | **Drag-and-drop + клики**, перемешанные правые элементы, частичный счёт |
| `OrderSteps.vue` | Упорядочить шаги | Drag-and-drop через HTML Drag API |
| `PromptComparator.vue` | **Кастомный** — выбрать лучший промпт | Показывает два промпта с «ответами AI», анализ качества |
| `PromptBuilder.vue` | Собрать промпт из блоков | Drag-and-drop по слотам (Роль/Контекст/Задача/Формат), среди блоков есть «шум» |
| `SpotTheHallucination.vue` | **Кастомный** — найти галлюцинацию | Кликабельные сегменты текста, цветовая легенда TP/FN/FP |

### Инфраструктурные

| Компонент | Назначение |
|---|---|
| `QuizRenderer.vue` | Диспетчер — принимает `component` и `props`, монтирует нужный компонент |
| `QuizRunner.vue` | Тест-режим: прогресс-бар, счёт в реальном времени, финальная сводка с рекомендациями |

### Два режима работы

**Inline-режим** (статьи): `QuizRenderer` встраивается в `ArticleView`. Каждый компонент работает автономно — нет общего счёта, можно сбросить и попробовать снова. Sticky-прогресс-бар в шапке статьи показывает «Упражнения: X / N».

**Тест-режим** (`/test`): `QuizRunner` управляет очерёдностью вопросов. Пользователь видит номер вопроса, текущий счёт, прогресс-бар. По итогу — сводка с процентом, количеством ошибок и персональной рекомендацией.

### Система оценки

- Каждое задание эмитит `{ correct: boolean, score: number, maxScore: number }`
- `MatchPairs` и `OrderSteps` — частичный счёт (3 из 4 пар)
- `FillTheBlank` — отдельный балл за каждый пропуск
- `SpotTheHallucination` — `score = найдено - ложные_срабатывания`, `maxScore = кол-во галлюцинаций`
- `QuizRunner` суммирует `score / maxScore` по всем вопросам

## Контент

**5 статей** с интерактивными упражнениями (inline-режим):

1. **Как работают большие языковые модели** — токены, контекстное окно, temperature
2. **Галлюцинации: когда AI уверенно врёт** — типы, причины, защита
3. **Анатомия эффективного промпта** — Роль + Контекст + Задача + Формат
4. **Токены и лимиты контекста** — что такое токен, контекстное окно, стратегии экономии
5. **Безопасность при работе с AI** — prompt injection, конфиденциальность, best practices

**Финальный тест** — 13 вопросов, все 8 типов компонентов.

## Развернуть локально

```bash
# 1. Установить зависимости
npm install

# 2. Запустить dev-сервер
npm run dev
# → http://localhost:5173

# 3. Собрать для продакшна
npm run build

# 4. Превью собранного
npm run preview
```

## Как попробовать

1. Откройте `http://localhost:5173` — главная с 5 статьями и динамической статистикой
2. Кликните на статью → читайте текст, отвечайте на упражнения inline; sticky-прогресс-бар показывает прогресс
3. В конце статьи — навигация к следующей или к финальному тесту
4. `/test` — финальный тест с 13 вопросами, прогресс-баром, финальной сводкой и рекомендациями
